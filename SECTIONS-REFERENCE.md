## Sections Overview Table

| Section Name | What It Is | What It Does | Use Case |
|-------------|-----------|--------------|----------|
| **Advanced Accordion** | Expandable/collapsible content panels | Allows users to click headers to reveal hidden content, with smooth animations | FAQs, service details, long content organization |
| **Advanced Image With Text** | Enhanced image + text layout | Combines imagery with descriptive text, often with advanced positioning and effects | About sections, feature highlights, storytelling |
| **Advanced Tabs** | Tabbed content interface | Organizes content into clickable tabs that switch displayed content | Services, product features, categorized info |
| **Animated Counters** | Number animation display | Shows statistics/numbers that animate (count up) when scrolled into view | Achievements, stats, milestones (e.g., "500+ Projects") |
| **Animated Stories** | Story-style content cards | Instagram/social media style stories with auto-progress or click navigation | Portfolio highlights, testimonials, quick updates |
| **Brands Section** | Logo showcase grid | Displays client/partner logos in a grid or carousel | Trust building, "Worked with" section |
| **Cards With Image** | Image-based card grid | Multiple cards with images, titles, and descriptions in a grid layout | Team members, services, portfolio items |
| **Carousel** | Sliding content display | Horizontal slider for images, testimonials, or content cards | Testimonials, galleries, featured work |
| **Comparison Table** | Side-by-side comparison | Table format comparing features, prices, or options | Pricing tiers, service packages, before/after |
| **Contact Form** | User input form | Collects visitor information (name, email, message) | Contact page, inquiry section |
| **Discount Bar** | Promotional banner | Thin bar (usually top) announcing deals or promotions | Sales, announcements, CTAs |
| **Discount Display** | Sale/promo showcase | Highlights discounts with countdown timers or badges | E-commerce, limited offers |
| **Download Promo Section** | Downloadable content promo | Promotes PDFs, apps, or resources with download CTA | Lead magnets, resource pages |
| **Dual Column With Image** | Two-column layout | Split layout with image on one side, content on other | About sections, feature explanations |
| **External Video** | Embedded video player | Embeds YouTube/Vimeo videos with custom styling | Demo videos, introductions, tutorials |
| **FAQ** | Frequently asked questions | Question/answer format, usually with accordion | Support, product info, service clarification |
| **FAQ Widget** | Compact FAQ module | Smaller FAQ component for sidebars or specific sections | Quick answers, product pages |
| **Featured Collections** | Grouped content showcase | Highlights curated groups of items/projects | Portfolio categories, service bundles |
| **Featured Products** | Highlighted items grid | Showcases select products/services prominently | Best sellers, key offerings |
| **Fixed Card Slider** | Sticky card carousel | Cards that slide while one element stays fixed | Feature tours, step-by-step guides |
| **HTML** | Custom code block | Allows insertion of custom HTML/CSS/JS | Custom widgets, embeds, unique elements |
| **Icon Testimonials** | Quote cards with icons | Testimonials with decorative icons and styling | Social proof, client reviews |

| **Info Expansion Section** | Expandable info blocks | Content that expands on hover or click for more detail | Dense information, specifications |
